# ⏱️ CHRONO — Precision Time Suite

A sleek and interactive **Clock + Stopwatch web application** built using pure HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://chrono-time-six.vercel.app/

---

## Features

* 🕒 **Real-time Analog + Digital Clock**
* 🌍 **Multi-Timezone Support** (Local, UTC, New York)
* ⏱️ **Advanced Stopwatch**

  * Start / Pause / Reset
  * Lap tracking
  * Best & Average lap stats
* ⚡ **Smooth Animations & UI Effects**
* 🎯 **Keyboard Controls**

  * `Space` → Start/Stop
  * `L` → Lap
  * `R` → Reset
* 📱 **Responsive Design**

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Custom UI + Animations)**
* **Vanilla JavaScript (No frameworks)**

---

## 🧠 Key Concepts Used

* DOM Manipulation
* requestAnimationFrame (for smooth updates)
* Event Handling
* Time calculations & formatting
* Dynamic UI rendering
